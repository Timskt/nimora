# ADR-007：正式产品与仓库名称采用 AsterPet

> 状态：Accepted  
> 日期：2026-07-17

## 背景

早期工作目录使用 `gptpet`，文档草案使用 DeskPet。前者将产品错误地限定为 GPT 应用，后者描述性过强，均不利于角色、自动化、创作者和开发者生态形成长期品牌。

## 决策

正式产品、GitHub 仓库、发布产物和新公共命名空间采用 **AsterPet**。公共 Schema 使用 `asterpet.*`，JavaScript 包使用 `@asterpet/*`，Rust crate 使用 `asterpet-*`。本地旧目录名不构成产品标识；历史归档不做内容重写。

## 结果

- 新代码、文档、包、应用标识和 UI 统一使用 AsterPet。
- 已经发布的公共标识不得静默重命名；当前尚未发布，可直接建立新命名空间。
- 后续增加商标、域名和商店名称检查，并通过 ADR 处理任何再次更名。
