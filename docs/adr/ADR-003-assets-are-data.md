# ADR-003：皮肤与角色资源默认是声明式数据

## 状态

Accepted

## 背景

皮肤是最常见的生态入口。允许资源包执行代码会显著提高预览、商店审核和版权治理风险。

## 决策

Character、Skin、Theme、Voice、Behavior 和 Interaction Pack 默认只能包含声明式数据和媒体。需要执行逻辑时必须发布独立 Skill，并单独展示权限。

## 后果

- 正面：资源可以安全预览、缓存、组合和热切换。
- 负面：复杂交互需要同时维护 Asset Pack 与 Skill。
- 缓解：提供语义事件和标准行为图，覆盖大部分无代码创作需求。
