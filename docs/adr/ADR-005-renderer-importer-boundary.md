# ADR-005：模型通过 Importer 与 Renderer Adapter 接入

> 状态：Accepted  
> 日期：2026-07-17

## 背景

序列帧、Live2D、Spine、glTF 和 VRM 在许可证、解析安全、动作模型与性能特征上差异巨大。若 Pet Runtime 直接依赖具体格式，核心语义、供应链风险和 UI 会被厂商实现绑架。

## 决策

Pet Runtime 只使用统一角色、动作和表达语义。第三方文件先由隔离 Importer 探测、校验和规范化，再由版本化 Renderer Adapter 加载。模型包默认不执行代码，专有运行时单独分发并遵守其许可证。

## 结果

- 正面：格式可替换、模型可安全预览、脚本可跨模型控制、核心可稳定降级。
- 代价：需要统一映射层、格式夹具、转换工具和 Adapter 契约测试。
- 缓解：允许格式特有能力作为可选扩展，但不能替代标准动作与回退路径。
